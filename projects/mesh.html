MESH Development

Overview

This simple HTML app highlights what the company does and who they are.

Purpose & Context

This site was built for a client who needed it in under 2 days for a charity auction.

Objective

Client needed a website that said a little bit about who they are and what they do.

Approach

Technologies: HTML, CSS

  • Created HTML shell
  • Added content
  • Added CSS styling
  • Hosted on GitHub Pages

Challenges

The challenging part of this project was the quick turnaround. There are a few changes I would like to make for next time. The animation of the paragraphs, I would change to trigger when the user scrolls down. The other change would be to have the company name stick to the top as it is scrolled past, and to not see the motto behind the name. This project was fun to experiment with animations.

Duration

This was built under a very tight deadline. The client needed a simple website up in 2 days.
